This is a commmand-line application that find items by keyword using FindingAPI. 

This sample was created using Microsoft Visual C# 2005.

To get started with this example Open the GeteBayOfficialTime.csproj file with visual studio.

You must edit the values in the web.config file and insert your Application ID which you obtained from http://developer.ebay.com. 

This application uses SOAP to make the request.
